# CSV_to_Excelツール利用案内

## １、このツールについて
本ツールは、CSVファイルをドラッグ＆ドロップするだけで、新しいExcelブックにCSVの内容を表示するためのツールです。

CSVファイルをExcelで通常どおり開いた場合、次のような問題が発生することがあります。
・日本語が文字化けする。
・電話番号、郵便番号などの先頭のゼロが消える。
・債権者番号などの長い番号が指数表記になる。
・Excelのインポート操作に手間がかかる。

本ツールでは、CSVファイルの文字コードを判定し、全ての列を文字列としてExcelに表示します。
元のCSVファイルは直接編集しません。

## ２、使い方
１、CSV_to_Excelフォルダを、デスクトップなど任意の場所に配置します。  
２、フォルダ内の「【ここにCSVをドロップ】to_Excel.bat」を確認します。  
３、開きたいCSVファイルを「【ここにCSVをドロップ】to_Excel.bat」にドラッグ＆ドロップします。  
４、自動的にExcelが起動し、CSVの内容が新しいExcelブックに表示されます。  

必要に応じて、「【ここにCSVをドロップ】to_Excel.bat」のショートカットをデスクトップに作成してください。

## ３、対応しているファイル
対応しているファイルは、拡張子が「.csv」のファイルです。
文字コードは、次の形式を想定しています。

・UTF-8
・UTF-8（BOM付き）
・Shift_JIS

## ４、重要な注意点

本ツールは、文字化け、先頭ゼロ落ち及び指数表記化を防ぐため、全ての列を「文字列」として読み込みます。

そのため、金額、数量、件数などの列は、開いた直後の状態では「数値」として集計できません。

「数値」として計算したい列がある場合は、次の「## ５、数値として計算したい場合」の操作を行い、対象列を数値に変換してください。

## ５、数値として計算したい場合

数値として計算したい列がある場合は、次の操作を行ってください。

１、数値に変換したい列又はセル範囲を選択します。  
２、Excel上部の「データ」タブを選択します。  
３、「区切り位置」をクリックします。  
４、「区切り位置指定ウィザード」が表示されたら、内容は変更せずに「完了」をクリックします。  

この操作により、選択した列がExcel上で数値として認識され、集計などが可能となります。

## ６、保存する場合

本ツールは、元のCSVファイルを直接編集せず、新しいExcelブックを作成し、CSVの内容を表示します。

保存する場合は、Excelの「名前を付けて保存」から、必要に応じてExcelブック形式（.xlsx）で保存してください。

CSV形式で保存し直す場合、Excel側の仕様により、文字コードや表示形式が変わる場合があります。

## ７、うまく動かない場合

次の点を確認してください。
・CSVファイルをドラッグ＆ドロップしているか。
・batファイルとps1ファイルが同じフォルダ内にあるか。
・Microsoft Excelがインストールされているか。
・CSVファイルの拡張子が「.csv」になっているか。
・CSVファイルを他のアプリで開いたままにしていないか。
・エラー画面に表示されたログファイルが残っているか。

エラーが発生した場合は、画面に表示されるログファイルの場所を確認してください。

## ８、ファイル構成
本ツールは、次のファイルを同じフォルダ内に配置して使用します。

・【ここにCSVをドロップ】to_Excel.bat
・CsvToExcel_backend.ps1
・README.md

batファイルとps1ファイルを別々の場所に移動すると、正しく動作しません。

## ９、使用上の留意事項

本ツールは、CSV確認作業を補助するためのものです。
重要なデータを扱う場合は、元のCSVファイルを保存した状態で使用してください。
大規模なCSVファイルを処理する場合は、Excelの起動や転記に時間がかかることがあります。
Excelの行数又は列数の上限を超えるCSVファイルは、Excel上に全て表示できません。
